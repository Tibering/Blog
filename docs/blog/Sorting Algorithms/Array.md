#### 数组的区间

数组下标默认从 `0` 开始，设区间范围为 `[L, R]` 或变体，以下为**数学定义 + Java 代码判断 + 遍历写法**极简总结。

#### 1. 左闭右闭 `[L, R]`

- 含义：包含 L、包含 R
- Java 判断：`idx >= L && idx <= R`
- 遍历：`for (int i = L; i <= R; i++)`

#### 2. 左闭右开 `[L, R)`

- 含义：包含 L、不包含 R
- Java 判断：`idx >= L && idx < R`
- 遍历：`for (int i = L; i < R; i++)`

#### 3. 左开右闭 `(L, R]`

- 含义：不包含 L、包含 R
- Java 判断：`idx > L && idx <= R`
- 遍历：`for (int i = L+1; i <= R; i++)`

#### 4. 左开右开 `(L, R)`

- 含义：不包含 L、不包含 R
- Java 判断：`idx > L && idx < R`
- 遍历：`for (int i = L+1; i < R; i++)`

### 数组划分

划分一个数组需要考虑考虑他的区间，一般我们是左开右闭，其他不用考虑，但是我们还是会看到。以下为左闭右开表示。

左开区间，意味着左边元素可以直接使用，然后右边最后一个元素需要减一。也就是说如果我将一个数组划成两个子数组。例如一个数组为[0, 9)。我们可以看到`index = 0` 为数组一个元素访问下标。而 `9`就是这个数组的`length`。也就是我们把数组分成两个，或者我们说删除第`i`个元素，那么这个`i` 将数组分为 前一个子数组为[0, i)，后一个数组为[i, 9)。那么访问就很简单了。

以左开右闭为例我们只需要知道一个`i`就是前一个数组的`length`。后一个数组第一个访问的下标。

这个在使用排序的十分常用，以下为数组使用示例

```java
// 假设 i=4
//
for (int k = 0; k < 4)
```

### 数组的缩进

我们说的缩进有两种，从最小下标开始缩进和从最大坐标开始缩进。

最小开始缩进就是`int i = 0; i++;`。从最大开始，假设我们从最大开始，则是使用`length` 去计算这个值`int j = length  - 1`。

最后还有一个点，就是`length`和元素个数， 元素个数 `n = length - 1`。
