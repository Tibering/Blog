# 排序算法

## 参考书籍

- [算法 第四版](https://book.douban.com/subject/19952400/)

### 读算法有感

算法第四版告诉我们，学算法不需要你知道什么高深的道理，只需要懂一门语言，然后会敲代码。也不需要纠结各种无聊时间复杂度计算，如果不会，则直接涉及程序进行计算。在保证以上的基础上，我们直接从排序开始。

当然，算法没有教你们怎么敲代码，在这篇总博客中，思考如何写代码。

正是因为没有强大的记忆力，和不想一次一次的重复记忆力。所以我们需要思维。
